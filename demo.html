üìù LRC File Format

The app uses the standard LRC (Lyric) format with timestamps:

[00:00.00]Song Title [00:03.45]Artist Name [00:07.20] [00:10.15]First line of lyrics [00:13.80]Second line of lyrics [00:17.25]Third line of lyrics

Format: [mm:ss.xx] where mm=minutes, ss=seconds, xx=centiseconds

üöÄ How to Use

Player Mode

  1. Open the app: Open index.html in your browser
  2. Load audio: Click "Choose Audio File" or drag an MP3/FLAC/WAV file
  3. Load lyrics: Click "Choose Lyrics File" or drag a .lrc file
  4. Play: Use the audio controls to start playback
  5. Enjoy: Watch the lyrics sync with your music!

Editor Mode

  1. Switch to Editor: Click the "Lyrics Editor" tab
  2. Load Audio (Optional): Load an audio file to help with timing
  3. Enter Lyrics: Type your lyrics in the text area (one line per verse)
  4. Mark Timestamps: Play audio and click "Mark Current Time" for each line
  5. Export: Add song info and click "Export .lrc File"
